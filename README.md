# AI 1on1 — 轻松看懂 AI

零基础交互式 AI 概念图解指南。用最简单的方式，快速了解 AI 世界里那些听起来很酷的词。

## 内容涵盖

- **基础概念**：LLM、Token、Context、Prompt
- **进阶概念**：Agent、Skill、MCP、RAG、Multi-Agent
- **公司与产品**：OpenAI、Anthropic、DeepSeek、阿里、字节等
- **Agent 产品对比**：OpenClaw、Manus、Claude Code、ChatGPT
- **AI 发展时间线**：从 Transformer 到 Multi-Agent 时代
- **趣味小测验**：检验学习成果
- **学习资源**：中国大陆可直接访问的教程与工具

## 特性

- 单文件 HTML，无需构建，浏览器直接打开
- 交互式 SVG 流程图，点击节点查看详解
- 词云动画、Token 可视化、Context Window 演示
- 暖色调设计，移动端适配

## 使用

直接打开 `index.html` 即可，或部署到任意静态托管服务。
